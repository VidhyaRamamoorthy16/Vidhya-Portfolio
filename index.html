ShopSphere

Intelligent API Gateway with Real-Time Monitoring

A full-stack e-commerce gateway system featuring a real-time Random Forest ML pipeline that inspects request patterns to block threats (SQLi, XSS, DDoS, brute force) with 90.21% accuracy at sub-millisecond speeds. Rather than binary IP blocking, an adaptive risk-scoring algorithm dynamically throttles suspicious traffic. Threat logs are streamed in real time via WebSockets to an isolated, read-only analytics dashboard, ensuring zero performance coupling with the primary gateway transaction flow. Deployed across a distributed multi-cloud stack including Vercel, Render, Upstash, and Supabase.
